## Files
| File | Purpose |
|------|---------|
| `index.html` | Main website (all pages in one file) |
| `CNAME` | Custom domain configuration |
| `_config.yml` | GitHub Pages settings |

## To Update the Website
1. Edit `index.html` locally
2. Commit and push changes to the `main` branch
3. GitHub Pages will automatically redeploy within 1–2 minutes

## Contact Form Setup
The contact form uses [Formspree](https://formspree.io).
After signing up at formspree.io, replace `YOUR_FORM_ID` in `index.html` with your actual form ID.
